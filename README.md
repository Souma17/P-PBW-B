# Proyek Web - Praktikum Git & GitHub

## Deskripsi
Repository ini dibuat untuk praktikum **Git & GitHub** pada mata kuliah Pemrograman Berbasis Web.  
Berisi latihan dasar mulai dari inisialisasi repository, branching, merge, hingga push ke GitHub.

## Langkah Pengerjaan
1. Instalasi Git dan konfigurasi user  
2. Inisialisasi repo lokal: `git init`  
3. Membuat file `index.html` dan `css/style.css`  
4. Commit pertama: `git add .` в†’ `git commit -m "init project"`  
5. Membuat branch `fitur-kontak` dan menambahkan `kontak.html`  
6. Merge branch `fitur-kontak` ke `main`  
7. Hubungkan ke GitHub dan push:  
   ```bash
   git remote add origin https://github.com/Souma17/P-PBW-B.git
   git branch -M main
   git push -u origin main
